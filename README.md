# 🚀 LeetMatric

LeetMatric is a simple dashboard that helps track your LeetCode progress by displaying solved problems and difficulty-based statistics in a clean UI.

## 📌 Features

- Track solved LeetCode problems
- View difficulty-wise progress (Easy / Medium / Hard)
- Visual progress circles
- Clean and minimal UI
- Fetch user statistics using LeetCode API

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- LeetCode API

## 📂 Project Structure

```
LeetMatric
│
├── index.html
├── style.css
├── script.js
└── assets
```

## ⚙️ How It Works

1. User enters their LeetCode username.
2. The application fetches data from the LeetCode API.
3. It calculates solved problems by difficulty.
4. Progress circles update dynamically based on solved questions.

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/richa948/Leetmatric.git
```

2. Open the project folder

```bash
cd Leetmatric
```

3. Open `index.html` in your browser.
